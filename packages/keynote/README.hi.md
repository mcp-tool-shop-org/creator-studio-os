# @क्रिएटर-स्टूडियो-ओएस/कीनोट

क्रिएटर स्टूडियो ओएस के लिए मुख्य विशेषताएं: स्लाइड डेक को स्वचालित करने, मार्कडाउन फ़ाइलें आयात करने, स्टोरीबोर्ड को FCPXML में बदलने, और विभिन्न फ़ॉर्मेट में निर्यात करने के लिए 56 उपकरण।

---

## स्थापित करें।

```bash
npm install @creator-studio-os/keynote
```

इसके लिए कीनोट (क्रिएटर स्टूडियो या स्टैंडअलोन संस्करण) और macOS 13 या उससे ऊपर का संस्करण आवश्यक है।

## यह पैकेज क्या करता है।

किसी भी ऐप्पल एप्लिकेशन की तुलना में, कीनोट ऐप्पलस्क्रिप्ट का सबसे विस्तृत उपयोग प्रदान करता है। यह स्लाइड शो बनाने, संपादित करने और निर्यात करने के लिए एक समृद्ध 'एसडीईएफ' शब्दकोश प्रदान करता है। `@creator-studio-os/keynote` इसमें सभी सुविधाएं शामिल करता है: 56 उपकरण जो स्लाइड के जीवनचक्र, टेक्स्ट, तालिकाओं, चार्ट, छवियों, ट्रांज़िशन, मशीन लर्निंग (एमएल) प्रभावों, निर्यात, और दो प्रकार के इंटरफेस (मार्कडाउन इम्पोर्ट + एफसीपीएक्सएमएल स्टोरीबोर्ड एक्सपोर्ट) को कवर करते हैं।

## उपकरण (56)

### एप्लिकेशन का जीवन चक्र।

| उपकरण। | विवरण। |
|------|-------------|
| `keynote_app_open` | कीनोट को सक्रिय करें। |
| `keynote_app_running` | जांच करें कि क्या कीनोट (Keynote) चल रहा है। |

### दस्तावेज़ का जीवन चक्र।

| उपकरण। | विवरण। |
|------|-------------|
| `keynote_open` | एक ".key" फ़ाइल खोलें; यह फ़ाइल का नाम लौटाता है (जो अन्य सभी उपकरणों द्वारा उपयोग किया जाता है)। |
| `keynote_close` | एक दस्तावेज़ बंद करें (वैकल्पिक रूप से, इसे सहेजें)। |
| `keynote_save` | एक दस्तावेज़ को सहेजें, यदि चाहें तो किसी अन्य फ़ोल्डर में। |
| `keynote_list_presentations` | सभी खुले दस्तावेज़ों की सूची दिखाएं। |
| `keynote_create_presentation` | एक नया, खाली प्रेजेंटेशन बनाएं। |
| `keynote_set_doc_size` | स्लाइड के आयाम निर्धारित करें (उदाहरण के लिए, 16:9 पहलू अनुपात के लिए 1920×1080)। |
| `keynote_set_kiosk_mode` | किओस्क डिस्प्ले के लिए ऑटो-प्ले, ऑटो-लूप और निष्क्रियता समय सीमा (आइडल टाइमआउट) को कॉन्फ़िगर करें। |

### विषय और विशेषज्ञ।

| उपकरण। | विवरण। |
|------|-------------|
| `keynote_list_themes` | उपलब्ध सभी थीमों की सूची दिखाएं। |
| `keynote_apply_theme` | किसी दस्तावेज़ पर एक थीम लागू करें। |
| `keynote_list_masters` | वर्तमान थीम में स्लाइड मास्टर लेआउट की सूची प्रदर्शित करें। |
| `keynote_set_slide_master` | एक स्लाइड के लिए मुख्य लेआउट निर्धारित करें। |

### स्लाइड प्रबंधन।

| उपकरण। | विवरण। |
|------|-------------|
| `keynote_list_slides` | सभी स्लाइडों की सूची बनाएं, जिसमें प्रत्येक स्लाइड का क्रमांक, शीर्षक और "छोड़ दिया" (स्किप) की स्थिति शामिल हो। |
| `keynote_get_slide` | एक स्लाइड के लिए, शीर्षक, मुख्य भाग, नोट्स और संक्रमण (ट्रांज़िशन) को पढ़ें। |
| `keynote_make_slide` | एक नया स्लाइड जोड़ें। |
| `keynote_delete_slide` | एक स्लाइड को हटाएं। |
| `keynote_duplicate_slide` | एक स्लाइड की प्रतिलिपि बनाएँ। |
| `keynote_reorder_slide` | एक स्लाइड को किसी अन्य स्थान पर ले जाएं। |
| `keynote_skip_slide` | किसी स्लाइड को "छोड़ा गया" के रूप में चिह्नित करें या उसे "छोड़ने योग्य" स्थिति से हटा दें। |

| `keynote_set_text_style` | किसी भी स्लाइड आइटम पर टेक्स्ट की शैली (फ़ॉन्ट, आकार, रंग) बदलें। |

| `keynote_set_presenter_notes` | एक स्लाइड पर प्रस्तुतकर्ता के लिए नोट्स सेट करें। |
| `keynote_extract_all_notes` | प्रत्येक स्लाइड से प्रस्तुतकर्ता के नोट्स और शीर्षकों को निकालें। |

### संक्रमण।

## उदाहरण

मार्कडाउन से एक प्रस्तुति बनाएं और इसे पीपीटीएक्स में निर्यात करें:

```json
// Tool: keynote_from_markdown
{
  "markdownPath": "/projects/brief.md",
  "masterMap": {
    "h1": "Title",
    "h2": "Section Header",
    "bullets": "Bullets"
  }
}

// Tool: keynote_export_pptx
{
  "documentName": "brief.key",
  "outputPath": "/projects/brief.pptx"
}
```

स्लाइड्स को प्रोरेस मूवी के रूप में निर्यात करें:

```json
// Tool: keynote_export_movie_advanced
{
  "documentName": "csos-showcase.key",
  "outputPath": "/projects/csos-showcase/out/slideshow.mov",
  "codec": "ProRes 4444",
  "width": 1920,
  "height": 1080,
  "frameRate": "29.97"
}
```

## मैकओएस आवश्यकता।

`@creator-studio-os/keynote` केवल मैकओएस के लिए है (`"os": ["darwin"]`)। एमएल टूल के लिए क्रिएटर स्टूडियो सदस्यता से कीनोट की आवश्यकता होती है। मानक टूल मैक ऐप स्टोर से मुफ्त स्टैंडअलोन कीनोट के साथ काम करते हैं।
